# 易控(Easycontrol)

## 简介

本软件基于开源项目Scrcpy，实现了其安卓客户端，并添加了一些功能，实现了安卓端控制安卓端。

## 功能特色

- 使用简单
- 支持音频传输
- 自动修改被控端分辨率，最大化利用主控端屏幕
- 良好的旋转支持
- 自动检测被控端屏幕状态，防止被控端熄屏
- 超快启动
- 低延迟
- 支持小窗显示与全屏显示
- 支持多设备同时连接
- 支持主控端以及多个被控端之间剪切板同步
