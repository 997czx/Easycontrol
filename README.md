## 简介
本软件基于开源项目Scrcpy，对其进行了大量魔改，实现了其安卓客户端，并添加了一些功能，实现了安卓端控制安卓端。

## 功能特色
- 使用简单
- 支持音频传输
- 启动迅速
- 低延迟
- 支持分辨率自适应
- 良好的旋转支持
- 支持自动控制被控端屏幕，控制更加便捷
- 支持小窗显示与全屏显示
- 支持多设备同时连接
- 支持多设备之间剪切板无感同步
- 支持物理键盘输入(不完善，目前仅全屏控制下支持，且需配合QQ输入法等输入中文)
